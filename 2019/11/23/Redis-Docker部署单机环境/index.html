docker拉取镜像及映射运行

# 拉取Redis最新镜像
docker pull redis 

# 获取redis配置文件（外网连接记得改bind 127.0.0.1）
wget http://download.redis.io/redis-stable/redis.conf
mkdir -p /usr/local/docker
mv redis.conf /user/local/docker

# 映射端口/配置文件启动
docker run -p 6379:6379 --name alice-redis -v /usr/local/docker/redis.conf:/etc/redis/redis.conf -v 
/usr/local/docker/data:/data -d redis redis-server /etc/redis/redis.conf --appendonly yes

docker查看Redis镜像运行情况

docker ps
查看运行中的容器是否存在刚才的Redis镜像

docker logs 容器名称/容器ID

查看容器启动日志

修改Redis配置

修改完redis.conf配置后重启容器 
docker restart 容器ID
